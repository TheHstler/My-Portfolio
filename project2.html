CAFE MENU - CSS

I was following a FreeCodeCamp tutorial on css.

For this project, I was following a CSS tutorial where I learned how to properly style a basic webpage. I built a simple cafe menu and used CSS to change things like the colors, fonts, and layout to make it look much more polished and professional. It was really interesting seeing how just a few small changes could completely transform the vibe of the page. Along the way, I also got a lot better at using classes, IDs, and linking external stylesheets. Overall, this project made me realize just how important good styling is for creating clean, user-friendly websites, and it definitely made me even more excited to keep learning.
