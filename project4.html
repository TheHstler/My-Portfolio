A full-stack web application for managing student records.

This project helped spark my passion for web development. While it started out as a CV layout using just HTML, it gave me a solid foundation in front-end structure and made me want to keep building.

I followed resources like freeCodeCamp, Codecademy, and tutorials on Roadmap.sh to improve my skills. Through trial, error, and repetition, I became more confident using HTML5 and CSS3. This particular project is a single-page CV build challenge, completed as part of the Roadmap.sh solutions. It's not just about how it looks - it’s about what it taught me: structure, layout, and writing clean, semantic HTML.

Since completing this, I’ve started working with more advanced tools like C#, ASP.NET Core, and SQL for building full-stack applications - including my Student Management App.
