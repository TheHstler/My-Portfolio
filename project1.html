city-Maze-Hackathon

This project is from my second Hackathon.
It was a group project and we created it using p5.js and AI tools, exploring ideas like body-controlled ping-pong and sign language-to-speech.
Ultimately, we developed a maze game set in real-world cities where players escape a zombie invasion on a real time map of a specific location in Bournemouth. You use your hand, which is tracked to move the zombie across the map. we did have some issues with this, however we perservered through trying different laptops and cameras. A camera would've been better to test this but we collectively agreed to just use the laptop as we had connectivity issues. However we all now know that next time a camera for the this technicality is better! In the end, we won the award for Best Technicality.
