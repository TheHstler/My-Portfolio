Smart Contract Analyser

🔍 Overview

At the Blockchain & Cybersecurity Hackathon at Bournemouth University,our team built a Smart Contract Security Analyser — a web-based tool designed to help developers identify vulnerabilities in Solidity smart contracts. The project combined static analysis with AI-powered insights to automatically detect common blockchain security issues and generate simplified audit summaries for easier understanding.

⚡ Features

  • Static Analysis Integration using Slither for Solidity vulnerability scanning
  • AI-powered summaries to explain technical risks clearly
  • Detection of reentrancy attacks, unsafe Ether transfers, and missing event logging
  • Built to support developers with limited auditing experience

💡 Why It Matters

Smart contract vulnerabilities have caused major blockchain losses, including attacks such as The DAO exploit. This project aimed to make blockchain security more accessible by helping developers understand risks before deployment.

🚀 What I Learned

  • Applied blockchain security concepts in a real hackathon environment
  • Worked with smart contract analysis tools and AI integration
  • Improved collaboration under a 5-day build deadline
  • Strengthened understanding of Solidity vulnerabilities and defensive development

Tools used: Slither, Solidity, OpenAI API, Web Development
